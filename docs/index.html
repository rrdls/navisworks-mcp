Navisworks MCP

Execute snippets C# focados dentro do Autodesk Navisworks a partir de um cliente MCP, usando um servidor local e um plugin carregado no Navisworks.

Começar GitHub
~/navisworks-mcp/session

navisworks-mcp start

Instalação local

Plugin Navisworks

O bundle é instalado em `%APPDATA%\Autodesk\ApplicationPlugins\NavisworksMcp.bundle`.

Servidor MCP

O launcher inicia o servidor local e mostra a URL `/mcp` para usar no cliente MCP.

Tunnel opcional

Cloudflare Tunnel ou ngrok podem publicar o endpoint local quando o cliente MCP exige URL pública.

primeiro teste

Pergunte ao Navisworks qual documento está aberto.

Depois de iniciar o servidor e executar o comando do plugin dentro do Navisworks, use um snippet pequeno para validar a conexão.
