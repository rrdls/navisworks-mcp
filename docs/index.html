Fluxo recomendado

Abra pelo launcher para carregar o addin corretamente.

O Navisworks não registra automaticamente este assembly .NET como o Revit fazia. O launcher abre o Navisworks e carrega o addin MCP explicitamente, de forma visível e previsível.

1

Instale

Execute NavisworksMcpSetup.exe do release navisworks-v0.1.0.

2

Inicie o MCP

Abra o launcher e clique em Start MCP para subir http://127.0.0.1:8000/mcp.

3

Abra o Navisworks

Clique em Open Navisworks. Se o Navisworks ja estiver aberto, use Load Addin.

4

Valide

Chame get_navisworks_context no cliente MCP e confirme "connected": true.

Uso ideal

Feito para coordenação e inspeção, não autoria BIM.

Contexto do documento

Leia título, caminho, versão da API e estado básico do documento ativo.

Seleção e propriedades

Inspecione itens selecionados, propriedades e dados de revisão do modelo federado.

Snippets C# focados

Execute trechos pequenos contra Autodesk.Navisworks.Api.Document com retorno em texto.

Limites atuais

O comportamento esperado é explícito.

  • Validado no Navisworks Manage 2024.
  • O carregamento do addin passa pelo launcher ou load-navisworks-addin.ps1.
  • Não cria paredes, famílias, níveis ou elementos nativos de autoria como no Revit.
  • Snippets devem ser pequenos, focados e preferencialmente de leitura.
